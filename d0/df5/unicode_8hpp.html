Converts the UTF-8 codepoint to their corresponding UTF-8 character sequence. Appends the sequence to end of a string.

Parameters
codepointThe UTF-8 codepoint that will be used in the conversion.
utf8The string that the converted codepoint will be append to.
errHolds the error message on failure. Use a reference to a local error instance. A value of nullptr will skip this process. Default value: nullptr.
Returns
bool - true on successful conversion else false.

Converts the entire UTF-8 string to a vector of UTF-8 codepoints.

Parameters
utf8The string that will be used in the conversion. The entire string will be converted.
byte_countHolds the number of bytes read. The dereferenced value will be 0 on failure. Use a reference to a local variable. A value of nullptr will skip this process. Default value: nullptr.
errHolds the error message on failure. Use a reference to a local error instance. A value of nullptr will skip this process. Default value: nullptr.
Returns
UTF8CodepointVector - the vector of codepoints. the vector will be empty on failure.
